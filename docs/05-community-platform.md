# 👥 Community & Affiliate Platform

> ระบบสมาชิก รีวอร์ด และ Affiliate ของ Zeus Sport

## ฟีเจอร์

### 👤 Member System
- ระดับสมาชิก: Bronze → Silver → Gold → Zeus
- สะสมคะแนนจากการซื้อ
- Exclusive benefits แต่ละระดับ

### 🎁 Reward System
- คะแนนสะสมแลกส่วนลด
- ของแถมพิเศษ
- Early access สินค้าใหม่

### 📢 Affiliate Program
- แฟนๆ ช่วยขายได้คอมมิชชั่น
- Unique referral link
- Track ยอดขาย real-time

### 🏆 Leaderboard
- Top Spender ประจำเดือน
- Top Promoter (Affiliate)
- รางวัลพิเศษสำหรับ Top 10
