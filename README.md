## 🚀 เริ่มต้นใช้งาน

### ความต้องการเบื้องต้น
- Node.js 18+
- npm หรือ yarn
- Git

### ติดตั้ง
```bash
# Clone repository
git clone https://github.com/taekhalifa420-tech/zeus-sport-ecosystem.git

# เข้าโฟลเดอร์
cd zeus-sport-ecosystem

# ติดตั้ง dependencies
npm install

# รัน development server
npm run dev
```
